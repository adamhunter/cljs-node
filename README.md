# Cljs

Node js compiler for cljs (ClojureScript).

## Installation

You must have Java 8 installed.

Add this line to your application's package.json:

```json
something: 'goes here'
```

And then execute:

    $ npm install

Or install it yourself as:

    $ npm install cljs
